I'm Hassan, a developer with a passion for bridging hardware and software. I build things ranging from iOS/macOS native apps using Swift and CryptoKit, to embedded systems running on Raspberry Pi boards, to computer vision tools that run in real-time.

I enjoy working across the full stack of a project — from the hardware wiring to the UI layer — and am particularly drawn to projects that solve real physical-world problems with clever software.

When I'm not coding, I'm usually tinkering with embedded electronics or exploring new ideas at hackathons.

HSN Gateway V2
Raspberry Pi WiFi Router / Security Gateway

A full-featured Pi-based WiFi gateway with a security-first design. Bridges an upstream Wi-Fi connection to a local hotspot, with real-time traffic graphs, a connected device list, per-device allow/pause/block controls, AdGuard Home ad-blocking, Suricata IDS/IPS, DNS-over-HTTPS, and GeoIP country-blocking via nftables. Includes a self-healing agent that auto-recovers the AP and DNS stack.
